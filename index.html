GitHub PR Label Query Builder

Build and apply label filters directly from the extension popup.

A simple, 100% free forever Chrome extension to compose Pull Request label searches on GitHub without memorizing search syntax.

Get started

Open the extension popup on GitHub → Pull requests, build your groups, and click Apply.

Install from Chrome Web Store

Features

  • Visual label groups with OR matching
  • Exclude labels with a single toggle
  • Preserves your existing search terms
  • Applies instantly to the GitHub PRs page
