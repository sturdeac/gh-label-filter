GitHub PR Label Query Builder

Build label queries for GitHub Pull Requests from the extension popup.

Version 1.0 • lightweight

Quick, no-nonsense tool for composing GitHub PR label searches. Create OR groups, exclude labels, and apply the query directly to GitHub from the popup, no memorizing search syntax.

What it does

Visual label groups Create OR groups of labels to match multiple labels at once.
Exclude labels Toggle a group as negative to exclude all labels in that group (produces -label:"...").
Preserves current search Existing query terms are preserved and merged with selected label groups.
Applies instantly Hit Apply in the popup and the PR search updates immediately on GitHub.

Example query

is:open is:pr label:"small,medium" -label:"wip"
Permissions
tabs — read active tab to know which page to update.
scripting — run a tiny script on the GitHub page to read labels and set the search URL.
Host — limited to https://github.com/* so the extension only runs on GitHub.

Get started

Open the extension popup while on GitHub → Pull requests. Build groups and click Apply.
Download
Install from the Chrome Web Store. This page is an overview — use the popup to run the tool.
